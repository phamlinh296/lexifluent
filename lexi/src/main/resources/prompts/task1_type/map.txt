## Task 1 Type: Map / Plan

Evaluation focus:
- Must describe changes to location or layout clearly (before vs. after, or described area)
- Overview must identify the most significant overall change (e.g., "the area was transformed from a residential to a commercial zone")
- Use location language accurately: "to the north of", "adjacent to", "in the centre", "replaced by"
- For before/after maps: describe what was there before and what replaced it
- Do NOT describe every minor detail — select the most significant changes

Language to reward:
- Location: "to the east/west/north/south of", "beside", "opposite", "surrounding"
- Change: "was demolished", "was converted into", "was replaced by", "remained unchanged", "was extended"
- Contrast: "whereas previously there was X, it is now Y"

Common weaknesses to flag:
- Describing location without change (for before/after tasks)
- Using cardinal directions incorrectly (confusing north/south)
- Mentioning every small detail without prioritising significant changes
- No overview paragraph
