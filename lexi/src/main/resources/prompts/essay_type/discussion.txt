## Essay Type: Discussion

The writer must present BOTH sides of an argument, then give their own view.

Evaluation criteria:
- Both views must be presented with equal weight and genuine development
- Writer's personal view must appear (typically in the introduction and conclusion)
- Penalise essays that omit one side entirely or give one side only one sentence
- Arguments for each side must be supported with reasoning or examples, not assertions
- Avoid "on the one hand... on the other hand" as the ONLY cohesive device — reward more sophisticated transitions

Structure expectation:
- Introduction: paraphrase topic + state both views will be discussed + own opinion
- Body 1: First view with development
- Body 2: Opposing view with development
- Conclusion: Restate own view clearly
